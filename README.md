# Hey, I'm Alex 👋

**CS @ University of Chicago** · Chicago, IL

Still finding what I love through projects and hacking. Love talking about my projects!

I build things at the intersection of **AI**, **3D visualization**, and **voice interfaces**. Most of my projects start from a question like *"what if you could talk to this?"* or *"what if you could see this in 3D?"*

---

### What I'm Into

- **AI Agents & LLMs** — building tools that observe, control, and evolve agent behavior
- **3D Visualization** — mapping data into explorable 3D spaces (Three.js, DeckGL, UMAP)
- **Voice Interfaces** — real-time speech-to-action pipelines with ElevenLabs & Claude
- **Hackathons** — rapid prototyping under pressure is my favorite way to learn
